# 両親

あなたたちには熱量が足りない。

いや、多分それは自分という失敗作を生み出してしまったことで無くしてしまっただけ。

人は失敗する生き物なのにそれを認められてあげていなかった。

教場前の部屋について。

もともとはおじいちゃんおばあちゃんが幸せそうに暮らしていた部屋だった。

僕という失敗作のせいでその部屋を分離させてしまった。

２人の心の溝になってしまった。

あなたたちの今は秋、これから寒い冬がやってきます。

どうか夫婦２人で仲良く、暖かく過ごせる場所になりますように。

# START DASH★

30年間抱え続けてきた重たい荷物を下ろすことができて、やっとスタート地点に立てたような気がします。

私たちの人生にはこれから暑い夏がやってきます。

きっとこれまでのような全力疾走ではすぐに疲れてしまうから、たまには木陰で休んだり遠回りを楽しんだりしながら、ゆっくり走って行きたいと思います。

私の失ってきた30年間という青春時代を、残り70年かけて取り戻して行きたいです。
人生100年かけて青春を堪能しようと思います。

今日という日が最高地点ではなく、毎日、明日はもっと楽しくなると思えるような、そんな人生を過ごしていきたいです。

そのためにはこれからもここに居る１人１人全員の力が必要です。

私には怒りの感情のコントロールが下手くそという大きな欠点があります。

今日のライブのように一体感を得るために使うべき拳という道具を別の用途で使ってしまわないか不安で不安で仕方ないです。

だから間違った使い方をしないで済むように、いつもそばで支えてくれる存在が必要です。

結婚してしまったらどこか遠くに行ってしまうような感覚があるかもしれないけど、私は絶対にそんなことにはしたくない。

だからこれからもずっと私たちのそばにいてください。

本日は本当にありがとうございました！
