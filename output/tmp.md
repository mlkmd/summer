# 音楽について

## 音楽の好きなところ

- 余白（多様な解釈の余地）があったりする。
- 下手くそな歌の方が胸を震わせたりする。
- 子供の頃は分からなくても、大人になって分かるようになったりする。
  - 逆も然り。
- 歌う側も作った時のその気持ちが分からなくなっていたり。
- アレンジするだけで全く違って聴こえたり。

## 音楽の嫌いなところ

- 圧縮しすぎで元のデータを復元できない
  - 言葉は冗長なくらいで丁度いい。
  - 元のドラフト情報から何から全て残した作品であってもいい。
    - Git 履歴

## ライブハウス

- 多様な考え方・感じ方をする人が集うところ
- 拳を暴力以外の用途で使うところ
- 思想が違くても一体感を感じられるところ
